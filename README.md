📌 GitHub Profile Finder
A simple React application that allows users to search GitHub profiles and view user details along with their public repositories.

✨ Features
Search GitHub users by username
Display user profile information
Show followers, following, and repository count
List public repositories in  grid
Loading state while fetching data
Error handling for invalid users
⌨Search on Enter key support

Tech Stack
React JS
JavaScript (ES6+)
Tailwind CSS
GitHub REST API
